Some scripts I made for [Azerothcore WoTLK Repack](https://www.ownedcore.com/forums/world-of-warcraft/world-of-warcraft-emulator-servers/wow-emu-general-releases/1040387-azerothcore-wotlk-repack-playerbots-individual-progression-32-other-modules.html#post4501356).

- **WoW Server.vbs** - for people who use [Windows Terminal](https://apps.microsoft.com/detail/9n0dx20hk701). It will open DB, auth server and world server in a single Windows Terminal window with 3 tabs. Then it will launch client if there's Wow.lnk in your server folder.
- **update_creature_respawn.py** - allows you to change spawn time for all mobs except bosses. You can set different multiplier for aggressive and neutral mobs.
- **update_quest_loot_chance.py** - sets drop chance for all quest-items to 100%. No more grinding for quest-items with 20% drop chance.
